# Smart_Text_Analyzer

features:
- Receive text from the user
- Calculation and display:
- Total number of characters (without spaces)
- Total number of words
- Number of sentences (based on the period.)
- Number of characters (English only)
- Find:
- The most frequent letter
- The most frequent word
- Operations on text:
- Reverse the entire text
- Review:
- Is the text a palindrome or not? (Ignoring spaces and punctuation marks)
- Replace the words:
- Change the words "bad" to "good"
- Change the words "hate" to "love"

- Language:
- Python
